## FAQ

### How do I get an API key?

Contact our team at support@ibnk.xyz to request an API key. Include information about your intended use case.

### What are the rate limits?

Default limits are:
- 100 requests per minute
- 20 requests per second burst

Contact us for higher limits if needed.

### Which networks are supported?

Currently supported:
- Arbitrum Sepolia (testnet)
- Base Sepolia (testnet)

More networks coming soon!

### Is there a testnet available?

Yes! All our current endpoints work on testnet networks. Perfect for development and testing.

### Can I use this in production?

The API is currently in testnet phase. Production mainnet support is coming soon.
