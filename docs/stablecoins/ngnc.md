# NGNC

## Overview

**`Official Site:`** https://www.linkio.africa/

The Nigerian Naira Stablecoin (*NGNC*) enabling cheap, transparent, decentralised, interoperable and frictionless settlements. LINK the issuer is a web3 infra payments infrastructure for Africa.

NGNC is always convertible at a 1:1 rate against the Nigerian Naira on LINK.
