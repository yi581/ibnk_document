# EURC

## Overview

**`Official Site:`** https://www.circle.com/en/euro-coin

[Euro Coin](https://www.circle.com/en/euro-coin) *(EURC)* is issued by Circle under the same full-reserve model as [USD Coin (USDC)](http://circle.com/usdc), a trusted dollar digital currency with more than $55 billion in circulation.

Designed for stability, Euro Coin is 100% backed by euros held in euro-denominated banking accounts so that it's always redeemable 1:1 for euros.

## How to Mint/Redeem

**`To mint/redeem:`** Visit [Circle](https://www.circle.com/en/euro-coin).
