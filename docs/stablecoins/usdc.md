# USDC

## Overview

**`Official Site:`** https://www.circle.com/en/usdc

[USDC](https://www.circle.com/en/usdc) was created by [Circle](https://www.circle.com/en/) and is considered the de facto settlement currency for DFX. USDC will be used across all DFX liquidity pools to ensure cross compatibility for all stablecoin swaps.

## How to Mint/Redeem

**`To mint/redeem:`** Visit [Circle](https://www.circle.com/en/usdc) or [Coinbase](https://www.coinbase.com/usdc)
