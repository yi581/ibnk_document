## Stablecoin Selection Criteria

iBnk only supports stablecoins that meet all of the following criteria:

### 1. Full Reserve Backing
Every token must be backed 1:1 by fiat currency held in regulated financial institutions.

### 2. Regular Audits
Independent third-party verification of reserves conducted regularly (at least quarterly).

### 3. Regulated Issuers
Issuers must operate under proper financial regulatory oversight (e-money license, trust charter, or equivalent).

### 4. Transparent Operations
Clear documentation of:
- Reserve holdings
- Audit reports
- Regulatory status
- Mint/redeem mechanisms

### 5. Technical Standards
- ERC-20 or equivalent token standard
- Secure smart contract implementation
- Active chain support and liquidity

## Adding New Stablecoins

iBnk continuously evaluates new fiat-backed stablecoins for potential inclusion.

### Nomination Process

To nominate a stablecoin for inclusion:

1. **Verify Eligibility** - Ensure it meets all selection criteria above
2. **Submit Documentation** - Provide:
   - Issuer information and regulatory status
   - Latest audit reports
   - Technical documentation
   - Liquidity and adoption metrics
3. **Evaluation** - iBnk team reviews submission
4. **Community Review** - Public comment period
5. **Integration** - Technical integration and testing
6. **Launch** - Mainnet deployment and announcement

Submit nominations to: stablecoins@ibnk.xyz
