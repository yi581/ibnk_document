### **Current Stage: Sandbox Environment**

The Incentive Program is currently active in **Sandbox**, where VSEs can:

* Connect wallets or platform accounts
* Test routing flows
* Try different settlement paths
* Review UI/UX
* Submit product feedback
* Report issues and suggest improvements

All Sandbox activity contributes to future incentive weighting.

### **Who Can Become a VSE?**

Anyone who passes verification can join as a VSE:

* Fintech companies
* Payment gateways
* Wallet providers
* Stablecoin issuers
* Developers
* Businesses experimenting with stablecoin settlement
* **Individual users with personal wallets (KYC required)**

VSE = any verified party that can interact with iBnk's routing layer.
