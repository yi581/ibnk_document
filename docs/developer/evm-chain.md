## EVM Chains

iBnk is currently deployed only on **EVM testnets**.
These networks are used purely for **Sandbox testing**, including routing simulation, API testing, and VSE onboarding.

***

## Current Testnet Deployments

* **Ethereum Sepolia**
* **Base Sepolia**
* **Arbitrum Sepolia**
* **Polygon Amoy**

These are temporary test environments for early partners to experiment with API calls and settlement flows.
